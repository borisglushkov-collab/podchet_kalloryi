# Design System — Wellness UI

Стиль: мягкий wellness / Yazio-like.

## Цвета (`AppColors`)

| Token | HEX | Назначение |
|-------|-----|------------|
| primary | `#6BCB77` | Кнопки, акценты |
| background | `#FFF9F0` | Фон экрана |
| surface | `#FFFFFF` | Карточки |
| streak | `#FF6B35` | Серия дней |
| water | `#4ECDC4` | Вода |
| protein / fat / carbs | синий / оранж / фиолет | макросы |

## Шрифт

Google Fonts **Nunito** через `google_fonts`.

## Навигация

3 вкладки: **Дневник** · **Аналитика** · **Профиль** (`MainShell`).
