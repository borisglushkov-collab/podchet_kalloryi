## Подключение весов

1. Закройте Futula Scale  
2. Разрешите Bluetooth + геолокацию (GPS)  
3. Встаньте на весы босиком  
4. Профиль → **Найти весы** → Health Scale / CF:E7…  
5. **Взвеситься**
